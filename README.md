# ✈️ 航班延误风险助手

输入航班号，实时查询延误风险 + 两地天气 + 目的地携带建议。

---

## 功能

- **实时航班状态** — 计划中 / 飞行中 / 已落地 / 取消 / 备降，显示登机口、航站楼、行李转盘
- **延误风险评级** — 综合航班延误数据和两地天气，给出🟢低 / 🟡中 / 🔴高三级评级
- **出发地 + 目的地天气** — 温度、湿度、能见度、风速、降水，天气描述全中文
- **具体出行建议** — 根据实际状况生成，如「雷暴建议提前到机场」「大型枢纽建议提前 2.5 小时」
- **目的地携带建议** — 按温度、降水、风速动态推荐带什么、不用带什么
- **全球航班覆盖** — 国内外主流航司均支持（国航、南航、东航、马航、新航、阿联酋等）

## 快速开始

### 1. 获取 API Key

前往 [AviationStack](https://aviationstack.com) 注册免费账号，获取 API Key（免费版每月 100 次）。

### 2. 安装依赖

```bash
pip install flask requests
```

### 3. 启动

```bash
AVIATIONSTACK_KEY=你的key python web/app.py
```

打开浏览器访问 http://127.0.0.1:5000

## 使用示例

输入航班号（支持各种格式）：

| 输入 | 说明 |
|------|------|
| `MH386` | 马来西亚航空 386 |
| `CA1234` | 中国国际航空 1234 |
| `EK201` | 阿联酋航空 201 |
| `CZ3101` | 中国南方航空 3101 |

## 风险评级逻辑

| 评级 | 触发条件 |
|------|---------|
| 🔴 高风险 | 航班取消/备降、延误 ≥ 45 分钟、雷暴/暴风雪、能见度 < 1 km、风速 > 65 km/h |
| 🟡 中等风险 | 延误 15–44 分钟、中大雨/小雪、能见度 1–3 km、风速 40–65 km/h |
| 🟢 低风险 | 无延误、两地天气晴好、能见度 > 3 km |

出发地和目的地天气均纳入评估——目的地雷暴同样可能导致飞机无法降落。
